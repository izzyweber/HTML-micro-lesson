What an .html file actually is

Three languages, one file, nothing to install

HTML the words and structure CSS how it looks JavaScript what it does, like the maths and filters

One file

It holds all three, plus tables, charts and images. There is nothing else to download.

Opens anywhere

Double-click and it runs in any browser, on any device, offline. No app and no licence.

Easy to share

Email the file, drop it in Teams, or host it at a link. That is the whole distribution step.

Objection · "Nobody else can edit it"

Anyone who can chat with an LLM can create or edit an HTML file

The fear is a black box that only its author can change.

The fix is to write the brief, the assumptions and the sources into the file, invisible to viewers but read instantly by the next person or the next AI.

Then editing is simple. Open it in any text editor, or hand it back to your favourite LLM and say what to change. You can even retype the words right here. Press E.

This deck carries its own brief, audience and sources, written in but never shown.
